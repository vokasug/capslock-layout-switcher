# CapsLock Layout Switcher

Легковесная утилита для Windows 10/11, которая превращает клавишу **Caps Lock** в переключатель раскладки клавиатуры.

Забудьте об неудобных `Alt+Shift` и `Win+Space`. Одно нажатие на Caps Lock — и язык меняется мгновенно. А поскольку стандартное поведение Caps Lock подавляется, вы никогда случайно не напечатаете ВЕСЬ ТЕКСТ ЗАГЛАВНЫМИ.

## Возможности

| Возможность | Описание |
|---|---|
| **Мгновенное переключение** | Нажмите Caps Lock для циклического переключения всех установленных раскладок. |
| **Подавление Caps Lock** | Индикатор никогда не загорается; текст остаётся в нижнем регистре. |
| **Системный механизм** | Использует тот же Windows API, что и `Alt+Shift`, работает в любом приложении. |

| **Иконка в трее** | Синяя иконка "CL" с меню: скопировать исходник или выйти. |
| **Автономный .exe** | Не требует Python, зависимостей и установки. |
| **Открытый исходный код** | Общественное достояние — используйте свободно в любых целях. |

## Быстрый старт

### Вариант 1: Готовый исполняемый файл

1. Скачайте `capslock_layout_switcher.exe` из раздела [Releases](../../releases).
2. Двойной клик для запуска. Никакой установки, Python или зависимостей.
3. (Опционально) Поместите файл в папку автозагрузки для запуска при входе в систему:
   ```
   Win+R → shell:startup
   ```
4. Нажмите **Caps Lock** для переключения языка.

### Вариант 2: Собрать из исходников

Требуется **Python 3.x** и **pip**.

```bash
pip install pyinstaller
python capslock_layout_switcher.py   # автоматически создаёт icon.ico
pyinstaller --onefile --noconsole --icon=icon.ico --add-data "capslock_layout_switcher.py;." capslock_layout_switcher.py
```

Исполняемый файл появится в папке `dist/`.

> **Примечание:** `--onefile` упаковывает всё в один автономный файл. При запуске он временно распаковывается — это нормальное поведение pyinstaller, в диспетчере задач будет видно два процесса до закрытия программы.

## Как это работает

### Переключение раскладки

Программа устанавливает низкоуровневый хук клавиатуры (`WH_KEYBOARD_LL`). При нажатии Caps Lock отправляет сообщение `WM_INPUTLANGCHANGEREQUEST` активному окну. Это тот же механизм, который Windows использует при `Alt+Shift` или клике по языковой панели — поэтому переключение **системное** и работает в любой программе.

Список установленных раскладок запрашивается через `GetKeyboardLayoutList` во время работы. Если у вас 2 раскладки — переключается между ними. Если 5 — циклически перебирает все 5.

### Подавление Caps Lock

После переключения раскладки программа проверяет состояние Caps Lock. Если он включён — синтезирует событие отпускания клавиши, чтобы немедленно выключить. Индикатор остаётся тёмным, регистр текста — нормальным.

### Защита от повторного запуска

При запуске программа создаёт именованный системный мьютекс (`CapsLockSwitcher_Mutex_7A3F`). Если другой экземпляр уже удерживает его, появляется поверх всех окон диалог:

> *"CapsLock Switcher уже запущен. Хотите завершить его?"*

- **Да** — сигнализирует запущенному экземпляру закрыться. Новый экземпляр **не запускается**.
- **Нет** — оставляет запущенный экземпляр работать. Новый экземпляр всё равно **не запускается**.

Работает надёжно независимо от того, включена ли иконка в трее.

## Настройка

Откройте `capslock_layout_switcher.py` в любом текстовом редакторе. Два флага в начале файла управляют поведением:

```python
ENABLE_LOGGING = 0      # 1 = писать capslock_switcher.log рядом с .exe
ENABLE_TRAY_ICON = 1    # 0 = работать полностью скрыто, без иконки в трее
```

| Флаг | `0` | `1` |
|---|---|---|
| `ENABLE_LOGGING` | Лог-файл не создаётся | Отладочный лог пишется в папку программы |
| `ENABLE_TRAY_ICON` | Нет иконки в трее, нет меню | Иконка в трее с меню по правому клику |

Изменения вступают в силу при следующем запуске программы.
