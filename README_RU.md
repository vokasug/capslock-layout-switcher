# CapsLock Layout Switcher

> Лёгкая утилита для Windows 10/11, которая превращает клавишу **Caps Lock** в переключатель раскладки клавиатуры, заменяя неудобные сочетания `Alt+Shift` и `Win+Space`. Стандартное поведение Caps Lock подавляется — больше никаких случайных ЗАГЛАВНЫХ БУКВ.

---

## Ключевые возможности

| Возможность | Описание |
|---|---|
| **Мгновенное переключение** | Нажмите Caps Lock — язык меняется мгновенно |
| **Подавление Caps Lock** | Индикатор не загорается; текст остаётся в нижнем регистре |
| **Системный механизм** | Использует тот же Windows API, что и `Alt+Shift`; работает в любых приложениях |
| **Защита от повторного запуска** | Предотвращает случайный повторный запуск с диалоговым окном |
| **Иконка в трее** | Синяя иконка «CL» с меню (копировать исходник / выход) |
| **Режим эмуляции Alt+Shift** | Опционально: эмулировать нажатие `Alt+Shift` вместо Windows API (переключается в меню трея) |
| **Автономный .exe** | Не требует Python, зависимостей или установки |
| **Открытый исходный код** | Public domain — свободное использование |

## Быстрый старт

### Вариант 1: Готовый исполняемый файл

1. Скачайте `capslock_layout_switcher.exe` из раздела [Releases](../../releases)
2. Двойной клик для запуска — установка не требуется
3. *(Опционально)* Добавьте в автозагрузку:
   ```
   Win+R → shell:startup
   ```
4. Нажмите **Caps Lock** для переключения языка

### Вариант 2: Сборка из исходников

Требуется **Python 3.x** и **pip**:

```bash
pip install pyinstaller
python capslock_layout_switcher.py   # автоматически создаёт icon.ico
pyinstaller --onefile --noconsole --icon=icon.ico --add-data "capslock_layout_switcher.py;." capslock_layout_switcher.py
```

Исполняемый файл появится в папке `dist/`.

> **Примечание:** `--onefile» упаковывает всё в один файл. При запуске временно распаковывается — это нормальное поведение pyinstaller; в Диспетчере задач будет два процесса до закрытия программы.

## Как это работает

### Переключение раскладки

Программа устанавливает низкоуровневую клавиатурную ловушку (`WH_KEYBOARD_LL`). При нажатии Caps Lock происходит одно из двух:

- **Эмуляция `Alt+Shift`** (по умолчанию) — синтезирует физические нажатия клавиш, как будто вы нажали их вручную. Работает надёжно даже когда метод Windows API даёт сбой.
- **Через Windows API** — отправляет сообщение `WM_INPUTLANGCHANGEREQUEST` активному окну. Тот же механизм, который Windows использует при клике по языковой панели.

Переключение между режимами — через меню иконки в трее: **Emulate Alt+Shift**.

Список установленных раскладок запрашивается через `GetKeyboardLayoutList`. Если у вас 2 раскладки — переключается между ними. Если 5 — циклически перебирает все 5.

### Подавление Caps Lock

После переключения раскладки программа проверяет состояние Caps Lock. Если включён — синтезирует событие отпускания клавиши, чтобы немедленно выключить. Индикатор остаётся тёмным; регистр текста не меняется.

### Защита от повторного запуска

При запуске создаётся именованный системный мьютекс (`CapsLockSwitcher_Mutex_7A3F`). Если другой экземпляр уже запущен, появляется поверх всех окон диалог:

> *«CapsLock Switcher is already running. Do you want to terminate it?»*

- **Yes** — сигнализирует запущенному экземпляру закрыться. Новый экземпляр **не запускается**.
- **No** — оставляет запущенный экземпляр работать. Новый экземпляр всё равно **не запускается**.

Работает независимо от настройки иконки в трее.

## Настройка

Откройте `capslock_layout_switcher.py` в любом текстовом редакторе. Три флага в начале файла управляют поведением:

```python
ENABLE_LOGGING = 0      # 1 = писать capslock_switcher.log рядом с .exe
ENABLE_TRAY_ICON = 1    # 0 = полностью скрытый запуск, без иконки в трее
EMULATE_ALT_SHIFT = 1   # 1 = эмулировать Alt+Shift, 0 = использовать Windows API
```

| Флаг | `0` | `1` |
|---|---|---|
| `ENABLE_LOGGING` | Лог-файл не создаётся | Отладочный лог в папке программы |
| `ENABLE_TRAY_ICON` | Нет иконки в трее | Иконка в трее с меню по правому клику |
| `EMULATE_ALT_SHIFT` | Windows API (`WM_INPUTLANGCHANGEREQUEST`) | Эмуляция физического нажатия `Alt+Shift` |

Изменения вступают в силу при следующем запуске.
